HOW TO USE THE OPTIMIZATION
================================================================================

For End Users:
  - Optimization is completely automatic and transparent
  - First search on a folder takes 20-24 seconds
  - Subsequent searches on same folder are instant (<10ms)
  - Different searches on same folder also instant (cache reuse)

For Developers:
  From backend:
    from backend.core.file_search_optimized import FileSearch
    searcher = FileSearch(use_cache=True)
    results = searcher.search_by_filename([folder], keywords)

  From frontend:
    from backend.core.file_search_optimized import FileSearch
    (Same interface - unified implementation)
