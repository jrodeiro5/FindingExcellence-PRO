================================================================================
STARTUP PROCEDURE
================================================================================

One-Time Setup:
  1. Install Python 3.12+ from python.org
  2. Install Ollama from https://ollama.com
  3. Download models:
     ollama pull qwen3:4b-instruct
     ollama pull phi4-mini
     ollama pull deepseek-ocr
  4. Run: uv-setup.bat (creates .venv and installs dependencies)

Daily Startup:
  1. Ensure Ollama is running (it runs as a service, usually automatic)
  2. Run: start.bat
     - Starts Backend (FastAPI on port 8000)
     - Waits 15 seconds for model warmup
     - Starts Frontend (CustomTkinter desktop app)
  3. Use the desktop application
  4. Run: stop.bat (or close windows) to shut down

Setup Verification:
  1. Run: check-dependencies.bat
  2. Verify all dependencies show [OK]
  3. If any show [MISSING], run: uv pip install -e .

================================================================================
STARTUP SCRIPTS EXPLAINED
================================================================================

start.bat
  - Checks virtual environment exists (.venv)
  - Verifies Ollama is running at localhost:11434
  - Checks all critical dependencies are installed
  - Starts Backend in new terminal window
  - Waits 15 seconds for model warmup
  - Starts Frontend in new terminal window
  - Shows API endpoints and features
  - Handles errors gracefully

stop.bat
  - Kills FastAPI process on port 8000
  - Closes FindingExcellence Backend window
  - Closes FindingExcellence Frontend window
  - Cleans up any remaining venv processes
  - Shows status of what was stopped
  - Total shutdown time: ~2-3 seconds

check-dependencies.bat
  - Verifies virtual environment
  - Tests all backend dependencies
  - Tests all frontend dependencies
  - Tests optional dependencies
  - Checks Ollama connectivity
  - Provides installation instructions if missing

================================================================================
TROUBLESHOOTING
================================================================================

Issue: "Ollama not found"
  Solution: Ensure Ollama is installed from https://ollama.com
           Check it's running: http://localhost:11434
           Ollama should auto-start on Windows

Issue: "Missing dependencies"
  Solution: Run: uv pip install -e .
           Then: uv pip install -r backend/requirements.txt
           Then: uv pip install -r frontend_desktop/requirements.txt

Issue: "Port 8000 already in use"
  Solution: Close other applications using port 8000
           Or run: netstat -ano | findstr :8000
           Then: taskkill /PID [pid] /F

Issue: "CustomTkinter not found"
  Solution: Run: uv pip install customtkinter==5.2.2

Issue: "Backend takes forever to start"
  Solution: First startup loads models (~30s)
           Subsequent starts are faster (~5s)
           Check Ollama is not also loading models

Issue: "Frontend won't connect to backend"
  Solution: Wait 15 seconds for backend warmup
           Check backend console for errors
           Verify port 8000 is listening: netstat -ano | findstr :8000
           Health check: http://localhost:8000/health

Issue: "Search is slow"
  Solution: First search scans filesystem (20-24s for large folders)
           Subsequent searches use cache (<10ms)
           This is normal and expected

INSTALLATION INSTRUCTIONS
================================================================================

Fresh Installation:

1. Clone or download the repository
   git clone https://github.com/jrodeiro5/FindingExcellence-PRO.git
   cd FindingExcellence-PRO

2. Install Ollama (if not installed)
   Download: https://ollama.com
   Run installer

3. Download AI models
   ollama pull qwen3:4b-instruct
   ollama pull phi4-mini
   ollama pull deepseek-ocr

4. Create virtual environment and install dependencies
   Run: uv-setup.bat
   Or manually:
     python -m venv .venv
     .venv\Scripts\activate
     uv pip install -e .

5. Verify dependencies
   Run: check-dependencies.bat

6. Start services
   Run: start.bat

7. Use the application
   Desktop UI opens automatically
   File search tab is default

8. Stop services
   Run: stop.bat
   Or close both terminal windows
