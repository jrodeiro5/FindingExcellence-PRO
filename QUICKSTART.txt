================================================================================
TROUBLESHOOTING
================================================================================

Q: "Ollama not found"
A: Ensure Ollama is installed from https://ollama.com
   Ollama should run as a background service automatically

Q: "Dependencies missing"
A: Run: check-dependencies.bat
   It will show what's missing and how to install

Q: "Port 8000 already in use"
A: Another application is using port 8000
   Either close the other app or wait a moment and try again

Q: "First search is slow"
A: YES! First search scans your entire drive (20-24 seconds)
   This is NORMAL and EXPECTED
   Subsequent searches use cache and are instant (<10ms)

Q: "Results table looks wrong"
A: If layout looks strange, resize the window
   CustomTkinter adjusts to window size

Q: "Files won't open on double-click"
A: Check that files still exist (they might have been deleted)
   If files are network drives, they may take longer to open
